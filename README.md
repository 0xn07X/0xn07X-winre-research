# 🧠 Windows System Recovery Research (WSRR)

Исследование механизмов восстановления Windows и их устойчивости к удалению критических компонентов.

---

## 📌 О проекте

Этот репозиторий — результат моего исследования поведения Windows 10/11 при отключении среды восстановления (WinRE) и удалении системного процесса csrss.exe.  

Цель — не навредить, а понять:  
- Как работает встроенная защита Windows.  
- Почему удаление одного файла может сделать систему неработоспособной.  
- Как действуют механизмы reagentc, takeown, icacls, rmdir.

> ⚠️ Проект строго учебный. Код тестировался только в изолированной виртуальной среде (VirtualBox).  
> НЕ запускайте на реальном компьютере. Это приведёт к необратимому повреждению ОС.

---

## 🔧 Технические детали

- Платформа: Windows 10/11 (x64)  
- Инструменты: WinAPI, C++, стандартные утилиты Windows  
- Ключевые команды:  
  reagentc /disable — отключение WinRE  
  rmdir /s /q C:\Windows\System32\Recovery — удаление среды восстановления  
  takeown + icacls + taskkill + DeleteFile — удаление csrss.exe
